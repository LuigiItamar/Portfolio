Ultimos Projetos:

Sistema Para Inserção de Vagas - RH

Desenvolvi uma solução integrada para o RH, permitindo o cadastro de vagas através de um painel administrativo. Os dados dos candidatos interessados são capturados e vinculados a páginas dinâmicas criadas automaticamente para cada vaga específica. O projeto utilizou o CMS Storyblok em conjunto com o framework Astro e framework Tailwind para estilização, garantindo alta performance, flexibilidade na gestão de conteúdo e uma experiência dinâmica e personalizada para os usuários.

Site Para Uma Universidade

O site tem como objetivo promover e vender cursos de pós-graduação e especializações voltadas para a área de tecnologia, facilitando o acesso a informações sobre programas, inscrições e benefícios. Desenvolvido com WordPress e Elementor, o site apresenta um design responsivo e intuitivo, priorizando uma experiência de usuário (UX) agradável e uma interface (UI) atraente. Além disso, foram implementadas práticas de otimização para mecanismos de busca (SEO) para aumentar a visibilidade online. Este projeto demonstra minha capacidade de integrar design, funcionalidade e estratégias de marketing digital para impulsionar a conversão de visitantes em alunos.

Revitalização Visual de Sistema de Fidelidade e Compras Online

Realizei a revitalização visual de um sistema de fidelidade e compras online para um mercado. A plataforma oferece descontos exclusivos para clientes cadastrados com CPF, exibe produtos em promoção e permite compras online diretamente no site. A atualização focou em modernizar o design, melhorar a usabilidade e garantir uma experiência responsiva para desktop, mobile e totens nas lojas físicas. Utilizando Bootstrap para estilização, entreguei uma interface mais atraente e intuitiva, mantendo a eficiência funcional do sistema.

Este projeto destaca minha habilidade em criar soluções de e-commerce eficazes, integrando design, funcionalidade e estratégias de marketing digital.
